## Vorbereitung

1. Python 3 installieren
2. PIP installieren
3. plotly installieren

   ```powershell
   pip install plotly==5.23.0
   ```
4. kaleido installieren

   ```powershell
    pip install -U kaleido
   ```
5. numpy installieren

   ```powershell
   pip install numpy
   ```
6. pytz installieren

   ```powershell
   pip install pytz
   ```
